# PersianChatFix - Luanti Mod

**PersianChatFix** is a Luanti mod that fixes Persian & Arabic chat, making it display correctly and readable.

## Features
- Converts all Persian and Arabic letters to their correct forms.
- Preserves all diacritics (Tashdid, Fatha, Damma, Kasra, etc.).
- Displays sentences in proper Right-to-Left (RTL) order.
- Compatible with other mods and the game UI.

## Requirements
- Requires a font that supports Persian/Arabic letters to display correctly.  
  Without a proper font, letters may appear disconnected or unreadable.

## Installation

### Windows / Linux / macOS
1. Download the mod ZIP from this repository.
2. Extract the ZIP file into your Luanti `mods` folder.
   - Example paths:
     - **Windows:** `C:\Users\<YourUser>\Luanti\mods\`
     - **Linux:** `/home/<YourUser>/Luanti/mods/`
     - **macOS:** `/Users/<YourUser>/Luanti/mods/`
3. Enable the mod in your world settings.

### Android
1. Copy the ZIP file into your Luanti `mods` folder using a file manager or via USB.
   - Example path: `/sdcard/luanti/mods/`
2. Enable the mod in your world settings.

### iOS
1. Use a file manager app (like Files) to place the ZIP in the Luanti `mods` directory.
2. Enable the mod in your world settings.
